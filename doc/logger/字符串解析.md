# 字符串解析



## 题目: 给你一串字符串，根据日志规则解析成固定格式

### 示例

输入

```cpp
 "%d{%Y-%m-%d %H:%M:%S}%T%t%T%N%T%F%T[%p]%T[%c]%T%f:%l%T%m%n"
```

输出 (注意每一个项都是一个三元组)

`std::vector<std::tuple<std::string, std::string, int> > vec;`

三元组解释: 

1. 第一位字符串对应下面的日志格式
2. 第二个字符串是输出格式(例如d 后面的一串对应的用户希望输出的时间格式)
3. 第三位数字，标识着是一个普通字符串，还是待解析文本。0代表普通字符串，1代表待解析文本

```cpp
d %Y-%m-%d %H:%M:%S 1
T  1
t  1
T  1
N  1
T  1
F  1
T  1
[  0
p  1
]  0
T  1
[  0
c  1
]  0
T  1
f  1
:  0
l  1
T  1
m  1
n  1
```

输出解释: 

```tex
%m 消息
%p 日志级别
%r 累计毫秒数
%c 日志名称
%t 线程id
%n 换行
%d 时间
%f 文件名
%l 行号
%T 制表符 tab
%F 协程id
%N 线程名称
默认格式 "%d{%Y-%m-%d %H:%M:%S}%T%t%T%N%T%F%T[%p]%T[%c]%T%f:%l%T%m%n"
```
